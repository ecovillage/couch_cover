# CouchCover

Tiny pieces of code to deal with a couchdb, handy for Sieben Linden specific use cases with a legacy db.

## Installation

Add this line to your application's Gemfile:

    gem 'couch_cover'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install couch_cover
